## Installation

1. Clone the repo and `cd` into it
2. `composer install`
3. Create a `pigeon` sql database and set your database credentials in the `.env` file.
4. Set your `APP_URL` in your `.env` file.
5. `npm install`
6. `npm run dev`
7. `php artisan storage:link`
8. `php artisan migrate --seed`
